# Snake Game

A classic terminal-based Snake game implementation using Python.

## Overview

This is an interactive Snake game where you control a snake on a 20×20 grid, eating food to grow longer while avoiding collisions with walls and yourself.

## Requirements

- Python 3.x
- Unix/Linux/macOS terminal (uses termios for keyboard input)

## How to Run

```bash
python3 play.py
```

## Controls

| Key | Action |
|-----|--------|
| **Arrow Keys** | Move snake (UP, DOWN, LEFT, RIGHT) |
| **W, A, S, D** | Alternative movement keys |
| **P** | Pause/Resume game |
| **R** | Restart game (when game over) |
| **Q** | Quit game |
